Quick Start Guide:

  1. Open DevTools:

    • Mac: Press Cmd + Option + I
  2. Go to "Console" tab
  3. Copy-paste the test commands below
  4. Watch results in console

Test Commands:

1. Check Backend Config

console.log('Backend URL:', API_CONFIG.API_BASE_URL)

2. Register New User

apiRegister({ name: 'Test User', email: 'test@example.com', password: 'password123', passwordConfirm: 'password123' })

3. Login (Optional - if already registered)

apiLogin({ email: 'test@example.com', password: 'password123' })

4. Get Current User

apiGetCurrentUser()

5. List All Profiles

apiListProfiles(1, 10)

6. Update Your Profile

apiUpdateProfile(1, { bio: 'Hello! I\'m a test user', skills: ['JavaScript', 'React', 'Node.js'] })

7. Get Education (Nested under Profiles)

apiGetProfileEducation(1)

8. Add Education (Nested under Profiles)

apiAddProfileEducation({ school: 'University of Albany', degree: 'Bachelor of Science', field_of_study: 'Computer Science', from_date: '2020-09-01', to_date: '2024-05-31' })

9. Delete Education (Nested under Profiles)

apiDeleteProfileEducation(1)

10. Get Education (Alternative Route)

apiGetEducation(1)

11. Get Experience (Nested under Profiles)

apiGetProfileExperience(1)

12. Add Experience (Nested under Profiles)

apiAddProfileExperience({ company: 'Tech Company Inc', title: 'Software Developer', location: 'New York, NY', from_date: '2024-06-01', current: true })

13. Delete Experience (Nested under Profiles)

apiDeleteProfileExperience(1)

14. Add Experience (Alternative Route)

apiAddExperience(1, { company: 'Tech Company Inc', title: 'Software Developer', location: 'New York, NY', from_date: '2024-06-01', current: true })

15. Get Experience (Alternative Route)

apiGetExperience(1)

16. Logout

apiLogout()
