# UAlbany Campus Portal (Static UI)

Static HTML/CSS UI for Albany Campus Portal platform. This project provides page layouts for authentication, profiles, posts, users and dashboard views, along with shared UI modules (navbar, chat). It is intended as a frontвЂ‘end template and does not include a backвЂ‘end or API integration.

Features The project interactivity includes:

- Updating profile info and image
- Adding and Deleting user's own profile only
- Liking and unliking posts, commenting in posts
- Sign up and Sign in

Deployed via [GitHub Pages](https://lindakovacs.github.io/UAlbany-Campus-Portal-Theme/)
